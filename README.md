# KanjiKit Frontend

Check it out here: [https://kanji-kit.vercel.app/]

Planned features: [https://trello.com/b/WV48B4hf/kanji-kit]

## Run locally

```git clone git@github.com:benbousquet/kanji-kit.git```

## Run frontend

```cd kanji-kit/frontend```

```npm install```

```npm run dev```

## Run API

```cd kanji-kit/api```

```bun install```

### rename .env and fill in

```bun migrate.ts```

```bun run dev```
