# KanjiKit Frontend

Check it out here: [https://www.kanjikitto.com/]

## Run locally

```git clone git@github.com:benbousquet/kanji-kit.git```

## Run frontend

```cd kanji-kit/frontend```

```npm install```

```npm run dev```

## Run API

```cd kanji-kit/api```

```bun install```

### rename .env and fill in

```bun migrate.ts```

```bun run dev```
